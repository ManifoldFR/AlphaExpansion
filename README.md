# AlphaExpansion

An implementation of alpha-expansion and graph cut in C++.

We use the [Boost Graph library](https://www.boost.org/doc/libs/1_72_0/libs/graph/doc/index.html).

## Building

If you are using the command line:
```bash
mkdir build
cd build/
cmake ../
make
```
